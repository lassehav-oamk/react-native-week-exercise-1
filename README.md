# Exercise: Basic React Native layout and styling

Task is to implement layouts and styles of the transaction confirmation view below. 
Implementation of any kind of logic is not needed. 

## Layout requirements

Here is the design of the layout, which you need to implement:
![Design](exerciseinfo-assets\layout-objective.png)

You need to implement the layout using React Native components and style the components to match the design as closely as possible. Use same text values as shown in the design. Automated tests will verify the correctness of your implementation.
